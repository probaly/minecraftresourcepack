In order to install these mods you'll first need the fabric loader from https://maven.fabricmc.net/net/fabricmc/fabric-installer/1.1.1/fabric-installer-1.1.1.exe
You'll need to install it on your client and select minecraft version 1.21.10 with the latest version of the loader, make sure to also create a profile with the installation, then click Install
<img width="454" height="274" alt="image" src="https://github.com/user-attachments/assets/cba6692e-7685-435b-a551-a006b6ccb58e" />
Then you want to open your Minecraft Launcher to locate the profile
<img width="1737" height="709" alt="image" src="https://github.com/user-attachments/assets/5ed4ce07-740c-455d-ac5c-4d09a9ec5b7a" />
The one you should be looking for should be the one with a custom icon, aka the one that usually has the name set as "Fabric Loader" 
<img width="935" height="82" alt="image" src="https://github.com/user-attachments/assets/107fc5da-696f-44ac-bb61-f24cdff4442d" />
Then you want to click the folder icon and find your mod folder inside of the installations folder
<img width="73" height="28" alt="image" src="https://github.com/user-attachments/assets/b0ed4fcb-7aa6-4a3c-99fc-ea5599785739" />
After you have located your mods folder, get your mods that you want to add (or the ones in the repository)
and drag them inside the mods folder
| Make sure to extract the mods zip file and drag ALL the mods INSIDE of the extracted zip file into the mods folder |
